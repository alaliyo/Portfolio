## 👋 소개

안녕하세요.  
저는 **가독성 높은 코드**, **기능 중심 구조 설계**, **유지보수하기 쉬운 프론트엔드 아키텍처**를 중요하게 생각하는 프론트엔드 개발자 김성원입니다.

3년간 다양한 웹 프로젝트를 수행하며 사용자 페이지, 관리자 페이지, 업무형 시스템을 개발해왔습니다.  
최근에는 **랑구스토리** 프로젝트에서 서비스 설계부터 사용자 홈페이지와 관리자 페이지 프론트엔드 개발까지 전반에 참여하며, Feature-Sliced Design 기반 구조와 공통 레이어 분리를 적용해 확장 가능한 구조를 만들어가고 있습니다.

관심사는 다음과 같습니다.

- 기능 중심의 프론트엔드 아키텍처 설계
- 공통 컴포넌트와 상태 흐름의 재사용성 향상
- API 계층 분리와 데이터 흐름 정리
- 렌더링 최적화와 유지보수 가능한 코드 구조
- 팀 내에서 일관된 개발 패턴을 공유할 수 있는 구조 만들기
